# SIPRU Mobile
Aplikasi mobile untuk Sistem Peminjaman Ruangan Kampus (SIPRU).

## Notes
Platform akan ditentukan pada tahap pengembangan berikutnya.

## Mobile Architecture
Mobile project menggunakan Flutter dengan struktur layered architecture:

- Presentation Layer (UI)
- Domain Layer (Model)
- Data Layer (Repository CRUD)

Integrasi API mengikuti endpoint backend ASP.NET Core.
